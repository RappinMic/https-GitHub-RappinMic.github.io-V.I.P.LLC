## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. No build process or dependencies required!

Alternatively, serve with any static file server:
```bash
python3 -m http.server 8080
# or
npx serve
```

## Design Philosophy

The site draws inspiration from Pablo Picasso's artistic periods:
- **Blue Period**: Melancholic blues and deep tones
- **Rose Period**: Warm pinks and earth tones
- **Cubism**: Geometric forms and abstract shapes

Each product and design element reflects this artistic heritage while maintaining a modern, e-commerce-ready interface.
